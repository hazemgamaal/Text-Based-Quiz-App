# Text-Based Quiz Application

A simple, interactive quiz app built in Python.  
Made as part of Software Engineering Phase 1.

---

Project Overview

This application:
- Displays multiple-choice questions
- Validates user input
- Calculates score
- Shows a final results summary
- Is fully modular (UI, Core Logic, Extra Features)
